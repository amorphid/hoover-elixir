# Hoover

A collection of Elixir functions that hoove (aka fetch) data from remote various sources.

## Installation

Add the following to you `mix.exs`:

    def application do
      [extra_applications: [:hoover]]
    end

    def deps do
      [{:hoover, "~> 0.1.0"}]
    end
